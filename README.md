# Hi, I'm Jonny 👋

I build AI and automation tools that people use every day. Most of it is internal systems for a NYC real-estate brokerage: CRM automation, underwriting, data pipelines, LLM assistants. That work lives in private repos, so what's public here is a small slice.

## Featured work

**[askbase](https://github.com/JonnyOctober/askbase)** — natural-language interface to any SQL database. Ask in plain English; an LLM introspects your schema, writes read-only SQL, returns formatted results. SQLite, Postgres, MySQL. Python.

**[bullshitometer](https://github.com/JonnyOctober/bullshitometer)** — paste text, a link, or a screenshot, get a claim-by-claim credibility score. React/Vite front end, Express proxy, web-search verification, SSRF-hardened unfurling. JavaScript.

**[convo2canvas](https://github.com/JonnyOctober/convo2canvas)** — a Discord bot that turns chat into art. GPT writes the prompt, Stable Diffusion renders it, real-time replies. Python.

**[jonnyoctober.com](https://github.com/JonnyOctober/jonnyoctober.com)** — my personal site. No framework, no build step, just careful CSS. HTML.
