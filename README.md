### 🛠️ Featured work

- **[askbase](https://github.com/JonnyOctober/askbase)** — a natural-language
  interface to any SQL database. Ask a question in plain English; an LLM
  introspects your schema, writes read-only SQL, and answers with formatted
  results. Pluggable across SQLite / Postgres / MySQL, with layered read-only
  safety. *Python.*

- **[bullshitometer](https://github.com/JonnyOctober/bullshitometer)** — paste
  text, a link, or a screenshot and get a claim-by-claim credibility score.
  React + Vite front end with an Express proxy that keeps the API key
  server-side, web-search verification, and SSRF-hardened link unfurling.
  *JavaScript.*

- **[convo2canvas](https://github.com/JonnyOctober/convo2canvas)** — a Discord
  bot that turns chat into generated art: GPT writes the prompt, Stable
  Diffusion renders it, the bot replies in real time. *Python.*

- **[jonnyoctober.com](https://github.com/JonnyOctober/jonnyoctober.com)** — my
  hand-built personal site. No framework, no build step, just careful CSS.
  *HTML.*
